# orcanote-agent-skills

虎鲸笔记的自定义 Agents 和 Skills。

## 项目目的

本项目为虎鲸笔记提供自定义 Agents 和 Skills，扩展 AI Agent 的能力，使其能够执行更复杂的任务和操作。

## 自定义 Agents

自定义 Agents 位于 `agents/` 目录中。

### 如何使用 Agents

请查阅您的 AI Agent 文档了解如何使用自定义 agents。

## 自定义 Skills

自定义 Skills 位于 `skills/` 目录中。每个 Skill 都是一个独立的功能模块，提供关于如何使用虎鲸笔记的特定知识。

### 包含的 Skills

- **excalidraw-whiteboard** - Excalidraw 白板绘图功能
- **mermaid-diagrams** - Mermaid 图表生成功能
- **orcanote-markdown** - Orca Note Markdown 处理功能

### 如何使用 Skills

1. 查阅您的 AI Agent 文档了解详细信息。
2. 通常将 skills 放入 `skills/` 目录中即可。
