# orcanote-agent-skills

Agents and skills for Orca Note.

## Project Purpose

This project provides custom Agents and Skills for Orca Note, extending AI Agent's abilities to perform more complex tasks and operations with Orca Note.

## Custom Agents

Custom Agents are located in the `agents/` directory.

### How to Use Agents

Please consult your AI Agent's documentation for how to use custom agents.

## Custom Skills

Custom Skills are located in the `skills/` directory. Each Skill is an independent functional module that provides specific knowledge about how to use Orca Note.

### Included Skills

- **excalidraw-whiteboard** - Excalidraw whiteboard drawing functionality
- **mermaid-diagrams** - Mermaid diagram generation functionality
- **orcanote-markdown** - Orca Note Markdown processing functionality

### How to Use Skills

1. Consult your AI Agent's documentation for details.
2. Usually placing the skills into the `skills/` directory would be sufficient.
